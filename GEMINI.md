# 機零壹科技網站重構專案 (G01 Website Refactor)

本文件記錄了機零壹科技股份有限公司網站的重構進度與結構。

## 專案目標
- **格式標準化**：將 Google Sites 導出的混亂格式轉化為乾淨的 HTML5 靜態網站。
- **美化視覺設計**：採用現代化 Vanilla CSS 與響應式佈局，建立專業工業視覺風格。
- **內容精優化**：徹底移除過時的「防疫隔板」內容，回歸專業加工與導光板核心業務。
- **統一資產管理**：集中圖片於 `assets/images/`，樣式於 `assets/style.css`。

## 目前檔案結構
- `index.html`：**首頁**。包含公司名稱、專業核心能力，已完全移除防疫隔板內容。
- `about.html`：**關於機零壹**。包含公司背景、經營理念與發展沿革。
- `products.html`：**產品與生產線**。展示加工規格、ISO 認證與產線展示。
- `assets/`
    - `style.css`：全站統一的現代樣式表（專業藍/工業灰配色）。
    - `images/`：存放所有網站使用的圖片資產（Logo、背景、產線照）。

## 重點更新記錄 (2026/02/23)
1. **內容重構**：刪除首頁及產品頁中所有防疫隔板的文字描述、訂購表單與輪播圖。
2. **樣式美化**：拋棄 Google Sites 冗餘腳本，手動編寫語義化 HTML 與簡約 CSS。
3. **路徑優化**：確保所有導覽連結與圖片路徑正確且易於維護。

## 技術備註
- 使用 Vanilla CSS 以確保輕量化與跨瀏覽器相容性。
- 支援響應式設計（RWD），適配手機與平板電腦。
