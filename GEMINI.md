# 機零壹科技網站重構專案 (G01 Website Refactor)

本文件記錄了機零壹科技股份有限公司網站的重構進度、結構與最新更新。

## 專案目標
- **格式標準化**：將 Google Sites 導出的靜態網站轉化為乾淨的 HTML5。
- **極致視覺感**：採用 Source Sans 3 + 思源黑體，建立大廠精密工業風。
- **動態交互**：加入滾動觸發動畫與自動圖片輪播，提升使用者體驗。
- **手機版優化**：針對行動裝置進行深度適配，解決導航換行與背景顯示問題。

## 目前檔案結構
- `index.html`：**首頁**。包含核心優勢、自動輪播圖片展示。
- `about.html`：**關於機零壹**。包含專業時間軸沿革、公司簡介（已修正為「自動生產線」）。
- `products.html`：**產品與生產線**。展示加工材質、技術規範（含輪播）與生產現場藝廊。
- `assets/`
    - `style.css`：全站樣式。含點陣紋理背景、捲動動畫、手機版暴力修復樣式。
    - `images/`：已全面改為**英文檔名**，確保行動裝置相容性。

## 重點更新記錄 (2026/02/27)
1. **視覺升級**：
    - 引入 **Source Sans 3** 與 **思源黑體** 字體組合。
    - 加入 **工業點陣 (Dot Matrix)** 背景紋理，增加材質深度。
    - 優化 Hero 區域，改用 135 度漸層遮罩並加入標題浮現動畫。
2. **功能強化**：
    - 實作 **Intersection Observer 捲動觸發動畫**（Reveal Effect）。
    - 在首頁與產品頁加入 **自動淡入淡出圖片輪播** 功能。
3. **相容性修復**：
    - **圖片更名**：將所有中文檔名圖片更名為英文（如 `factory-gate.jpg`, `panel.jpg` 等）。
    - **手機版修復**：解決了導航列文字換行、大標背景圖消失以及輪播圖塌陷問題。
4. **內容微調**：
    - 術語修正：「電視導光板」 -> 「顯示器導光板、擴散板」。
    - 移除冗餘資訊：刪除頁尾的負責人與統編資訊，統一三頁頁尾。

## 技術備註
- 使用 Vanilla CSS 與原生 JS，無外部依賴，加載速度極快。
- 圖片資產建議維持 1200px 以上寬度以確保 Retina 螢幕顯示清晰。
- 官方 Logo (Favicon)：`assets/images/g01-logo-square.jpg`。
